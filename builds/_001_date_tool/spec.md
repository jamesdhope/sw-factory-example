# Technical Specification: Date Fetching Tool

## Objective
Develop a simple Python script to fetch and display the current date and time, suitable for Python developers needing a quick utility.

## Functional Requirements
1. **Fetch DateTime**: The script should retrieve the current date and time.
2. **Output Format**: Display the date and time in a human-readable format, specifically `YYYY-MM-DD HH:MM:SS`.

## Non-functional Requirements
1. **Code Simplicity**: Prioritize simplicity and readability of code.
2. **Compatibility**: Ensure the script runs on Python 3.
